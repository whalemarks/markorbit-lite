## Purpose

Foundation Sprint 0.0 establishes implementation governance for MarkOrbit Lite before application development starts.

## Architecture Decisions

- Lite is a clean-start implementation.
- V1 may inform business thinking but is not inherited as implementation architecture.
- Lite is positioned as the first usable application layer for Mo capabilities.
- Mo Brain is reserved and disabled until a future authorized integration.
- Workflow Runtime is reserved, not implemented.
- User workspace data is private by default.
- Future Runtime compatibility is preserved through explicit boundaries.
